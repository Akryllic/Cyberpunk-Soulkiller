## Installation & System Requirements

### Q: Can I install Soulkiller on an external drive or HDD?
**A:** No. This modlist requires an NVMe Gen 3 or newer drive. External drives and HDDs are not supported and will cause severe performance issues and crashes.

### Q: Do I need 16GB of RAM?
**A:** At least 16GB of RAM is required for optimal performance. The modlist is memory-intensive, so more RAM will improve stability.

### Q: Can I install the game in Program Files?
**A:** No. You must install Cyberpunk 2077 in a separate location like `C:\Games`. Installing in Program Files will cause permission issues.

### Q: What is RedMod and why do I need it?
**A:** RedMod is Cyberpunk 2077's official modding utility. It's required for full mod functionality and script loading. Download it through Steam or from the official source.

### Q: Do I need to configure the pagefile?
**A:** Yes. This step is mandatory regardless of your RAM capacity. Larger modlists require significant virtual memory to prevent crashes.

## Performance & Graphics

### Q: How much VRAM do I need?
**A:** 12GB+ of VRAM is strongly recommended for 1080p gameplay. Higher resolutions (1440p, 4K) require more VRAM.

### Q: Should I enable ray tracing with this modlist?
**A:** Soulkiller is built with advanced graphics in mind. Ray tracing is supported, but your hardware must be capable. Test performance and adjust DLSS settings accordingly.

### Q: What's the difference between Preset B and other Frame Generation presets?
**A:** Preset B offers a balance between performance and visual quality. Experiment with presets based on your hardware for optimal results.

### Q: Why should I set the Shader Cache to 10GB?
**A:** A larger shader cache reduces compilation stutters and improves frame time consistency as the game compiles new shaders.

## Troubleshooting

### Q: My antivirus is interfering with installation. What do I do?
**A:** Add the modlist folder, game executable, and redmod executable to your antivirus exclusion list. You may also need to temporarily disable the antivirus during installation.

### Q: I'm getting crashes or low FPS. What should I check?
**A:** 
1. Verify your pagefile is configured correctly (40960 MB initial and max)
2. Ensure your shader cache is set to 10GB (NVIDIA only)
3. Check that all prerequisites (.NET, Visual C++) are installed
4. Verify you're using an NVMe Gen 3+ drive
5. Monitor your VRAM usage and reduce graphics settings if needed

### Q: The game won't start or loads very slowly.
**A:** This is typically caused by:
- Insufficient pagefile configuration
- Game installed on an unsupported drive (HDD/external SSD)
- Missing or outdated .NET or Visual C++ redistributables
- Antivirus interference

### Q: Can I use this modlist with remote desktop or cloud gaming?
**A:** No. Remote PC services and cloud gaming apps are not supported. You must run the game locally.
